Informations

Objectif du site

Task Master est une application web de gestion de tâches gamifiée conçue pour vous aider à organiser vos objectifs et suivre votre progression. L'application utilise un système de points, de rangs et de récompenses pour rendre la productivité plus engageante et motivante.

Traitement des données

Vos données sont traitées de manière sécurisée et respectueuse de votre vie privée :

  • Stockage local : Toutes vos tâches et vos données de progression sont stockées localement dans votre navigateur (localStorage).
  • Aucun serveur : Aucune donnée n'est envoyée à des serveurs externes. Vous avez le contrôle total de vos informations.
  • Pas de suivi : Nous ne collectons pas de données personnelles, d'adresses IP ou d'informations de suivi.
  • Suppression : Vous pouvez supprimer toutes vos données à tout moment via les paramètres de configuration.
